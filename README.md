# Penny Tracker
Приложение за следене на разходи, създадено с **React** и **Supabase**.

## Функции
### 1.Регистрация и логин
-  Регистрация с имейл и парола
-  Вход с имейл и парола
-  Вход/регистрация чрез Google OAuth
-  Забравена парола → изпращане на имейл с линк за нова
  
 ### 2.Управление на потребител
- Промяна на парола
  
### 3.Управление на разходи 
-  Добавяне на нов разход (име, сума, категория, дата)
-  Изтриване на записи
-  Преглед на всички разходи за текущия потребител
  
## Технологии
- React
- MUI (Material-UI) – за готови UI компоненти
- Formik – за управление на форми и валидация
- Supabase (Auth + Database)
